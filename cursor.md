## Personal Task & Notes Manager

Step 1: Create Backend (Flask App)

Prompt 1: 
```
Create a beginner-friendly Flask app in Python.

Requirements:
- App name: task manager
- Use SQLite database
- Table: tasks (id, title, description, completed)
- Routes:
  - GET / → show all tasks
  - POST /add → add new task
  - POST /complete/<id> → mark task completed
- Keep code simple and well commented
```

prompt 2:

```
Add a delete task feature.

Requirements:
- Delete button next to each task
- Confirmation before delete
- Update database accordingly

```
